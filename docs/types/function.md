# Type: `function`

A set of code that can be "called" to run it.

```
func example() {
    bark("this code is only ran when we call the function")
}

example(); # call the function
```

# Operations

## `isfunction(value)`
Returns `true` if `value` is a function.

```
bark(isfunction(push)); # true!
```
