# Chapter 1.2: Install & Setup

GLang is available for macOS and Windows. You can install it from [here](/docs/install/).

We also suggest using an Integrated Development Environment (IDE) to write code. An IDE is your companion for development, like a super advanced version of Google Docs. For those who want to jump right into coding, a plain text editor alongside a terminal window works well.
